# **App Name**: CourseWise

## Core Features:

- Course Material Upload: Teachers can upload PPT, DOC, PDF, and MD course materials.
- Learning Objective Definition: Teachers can define learning objectives, skills, and trajectories for each course.
- AI-Powered Socratic Chat: Students can chat with course content for a socratic learning experience, with strict compliance to the provided materials enforced as a tool by the LLM.
- Personalized Assessments: Students receive personalized assessments based on their learning paths.
- Teacher Reporting: Teachers receive reports on student usage and engagement to inform interventions.
- Content Storage: Teachers store PPT, DOC, PDF, and MD course materials on the cloud database

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey a sense of learning and intellectual depth.
- Background color: Light grey (#ECEFF1), a desaturated version of indigo to avoid distracting the student.
- Accent color: Soft violet (#9575CD), for subtle contrast on interactive elements.
- Body and headline font: 'Inter', a sans-serif with a modern, objective look.
- Use consistent, professional icons for course navigation and material types.
- Clean and structured layout to facilitate easy navigation and content consumption.
- Subtle transitions and animations to provide feedback on user interactions.
